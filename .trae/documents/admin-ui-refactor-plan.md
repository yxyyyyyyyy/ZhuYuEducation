# 管理员端 UI 改造计划

## 需求概述

1. **题库总览**：左侧分类树太长 → 改为右侧弹窗（Drawer）选择分类，页面主体只显示一级标题标签
2. **知识图谱改名**：导航和页面标题改为「教材管理」
3. **教材管理**：选择顺序改为先选年级再选学科；知识点树改为三级标题（一级：如遗传学，二级：如生物分走，三级：如基础知识/核心方法）

---

## 步骤一：题库总览 — 分类改为右侧弹窗

### HTML 改动 (`admin.html`)

- **删除**左侧 `category-pane` 区域（`question-layout` 中的左侧 panel）
- **删除** `.question-layout` 的双栏 grid 布局，改为单栏
- **在题库表格上方**添加一级分类标签栏（横向排列的 tag/chip），点击某个一级分类时打开右侧弹窗
- **新增**右侧弹窗 HTML 结构（Drawer），包含：
  - 半透明遮罩
  - 右侧滑出面板（宽度 360px）
  - 面板内：二级分类列表 → 点击展开三级知识点列表
  - 选中某项后关闭弹窗并筛选

### CSS 改动 (`admin.html` style 区域)

- 新增 `.drawer` 样式：固定定位、右侧滑入动画
- 新增 `.drawer-mask` 样式：半透明遮罩
- 新增 `.category-tag` 样式：一级分类标签（pill 形状）
- 修改 `.question-layout` 为单栏

### JS 改动 (`admin.js`)

- 新增 `openCategoryDrawer()` / `closeCategoryDrawer()` 函数
- 修改 `renderQuestionCategoryTree()` → 拆分为两个函数：
  - `renderQuestionCategoryTags()`：渲染页面上一级分类标签
  - `renderCategoryDrawerContent()`：渲染弹窗内二三级树
- 修改 `bindEvents()` 中相关事件绑定
- 新增弹窗打开/关闭事件

---

## 步骤二：知识图谱 → 教材管理 改名

### HTML 改动

- 侧边栏导航：`知识图谱` → `教材管理`，图标改为 📖
- 页面标题：`知识图谱配置` → `教材管理`
- 页面描述：更新措辞

### JS 改动

- 无逻辑改动，仅文案

---

## 步骤三：教材管理 — 先选年级再选学科 + 三级标题

### HTML 改动

- 教材选择区域：新增年级下拉框 + 学科下拉框（级联：先选年级，再选学科）
- 知识点操作表单中的层级选项文案更新：
  - 一级目录（学科）→ 一级标题（如：遗传学）
  - 二级目录（年级/学段）→ 二级标题（如：生物分走）
  - 三级目录（知识点）→ 三级标题（如：基础知识、核心方法）
- 新增年级输入框和学科输入框用于新建教材

### JS 改动

- 新增年级/学科级联逻辑：选择年级后筛选可选学科
- 修改 `loadTextbooks()`：按年级→学科分组渲染下拉选项
- 修改 `createTextbook()`：先填年级再填学科
- 知识点树渲染中更新层级显示文案

---

## 涉及文件

| 文件 | 改动类型 |
|------|---------|
| `app/templates/admin.html` | HTML 结构 + CSS 样式 |
